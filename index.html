Como Formatar e Reinstalar o Windows com Pendrive USB e Rufus

Aprenda passo a passo como dar uma nova vida ao seu PC, remover vírus e reinstalar o Windows 10 ou 11 de forma segura e profissional.

Começar Agora

Por que e Quando Formatar seu Computador?

Bem-vindo ao guia definitivo para formatar seu computador com Windows. Se você chegou até aqui, provavelmente deseja dar uma nova vida ao seu PC, resolver problemas crônicos de desempenho, remover vírus persistentes ou simplesmente reinstalar o sistema operacional do zero para ter aquela sensação de computador novo.

Neste guia completo, vamos utilizar um pendrive USB e um programa gratuito chamado Rufus para criar uma mídia de instalação do Windows. Depois, mostraremos como configurar seu computador para iniciar a partir desse pendrive e realizar a instalação limpa do Windows 10 ou Windows 11.

Atenção Importante: Formatar um computador apaga TODOS os dados do disco rígido onde o sistema será instalado. Certifique-se de fazer backup de todos os seus arquivos importantes (documentos, fotos, vídeos) para um local seguro antes de começar!
Capítulo 1

Preparativos Essenciais

1. O Pendrive USB

Vou deixar meu link de afiliado da Shopee caso você precise de um pendrive confiável para não correr o risco de comprar em sites maliciosos. Se precisar, basta clicar na imagem do Pendrive acima para ser direcionado à Shopee.

Você precisará de um pendrive com no mínimo 8GB de capacidade. Para o Windows 10 ou 11, recomendamos um dispositivo de 16GB ou mais para garantir que os arquivos caibam sem problemas. Lembre-se: tudo o que estiver no pendrive será apagado durante o processo.

Especificações do Pendrive:

  • Capacidade mínima: 8GB (Recomendado: 16GB ou superior)
  • Velocidade: USB 3.0 (conector azul) para uma instalação muito mais rápida

Uma imagem ISO é o arquivo compactado oficial que contém todo o sistema operacional. Baixe sempre direto do site da Microsoft pelos links acima.

Passo a Passo com a Ferramenta da Microsoft:

  1. Baixe a ferramenta clicando em "Baixar a ferramenta agora" no site oficial.
  2. Execute o arquivo MediaCreationTool.exe e aceite os Termos de Licença.
  3. Escolha a opção "Criar mídia de instalação (pendrive, DVD ou arquivo ISO)".
  4. Selecione o idioma, a edição e a arquitetura (sempre escolha 64-bits).
  5. Na tela de escolha de mídia, selecione "Arquivo ISO" e escolha uma pasta no seu PC para salvar.

Nota Técnica: Certifique-se de marcar a arquitetura correta de 64 bits para aproveitar toda a memória RAM do seu computador.
Capítulo 2

Baixando e Configurando o Rufus

O Rufus é um software leve e gratuito que vai transformar seu pendrive comum em uma mídia de inicialização (boot) oficial do Windows.

Como Baixar o Rufus Oficial

  1. Acesse o site oficial: https://rufus.ie/pt/
  2. Role até a seção de Download.
  3. Baixe a versão mais recente (ex: rufus-4.x.exe ou a versão portátil).

Criando o Pendrive Bootável

Configuração Correta do Rufus:

  1. Insira o pendrive na porta USB do computador. -> Verifique se ele foi reconhecido.
  2. Abra o Rufus. Em "Dispositivo", selecione o seu pendrive.
  3. Em "Seleção de boot", clique no botão SELECIONAR e escolha o arquivo ISO do Windows que você baixou.
  4. Em "Esquema de partição", escolha:
    • GPT (UEFI): Para computadores modernos (Recomendado).
    • MBR (BIOS): Apenas se o seu computador for muito antigo.
  5. Clique em INICIAR. O Rufus vai exibir um alerta avisando que tudo no pendrive será apagado. Clique em OK e aguarde a barra verde atingir 100%.

    6. Aguarde o status mudar para "PRONTO"

Capítulo 3

Configurando a BIOS para Dar Boot pelo Pendrive

Com o pendrive pronto, reinicie o computador mantendo o pendrive conectado. Você precisará acessar a placa-mãe para ordenar que o PC inicie pelo pendrive antes de ler o HD interno.

Teclas comuns para acessar a BIOS/Boot Menu:

Assim que a tela acender ao ligar o PC, pressione repetidamente as teclas: F2, F12, Delete (Del) ou F11.

Dentro da BIOS, navegue com as setas do teclado até a aba Boot e coloque o seu dispositivo USB/Pendrive como a Opção de Boot #1 (Option #1). Salve as alterações pressionando F10 e saia.

Capítulo 4

Instalação Passo a Passo do Windows

Se a BIOS estiver configurada certa, a tela de instalação clássica do Windows vai carregar na inicialização.

1. Tela Inicial de Idioma

Defina o idioma como Português (Brasil) e o teclado para o padrão Português (Brasil ABNT2) (que possui a tecla Ç). Clique em Avançar e depois em Instalar Agora.

2. Chave de Ativação

Na tela que pede a chave do produto, caso esteja reinstalando na mesma máquina, clique em "Não tenho a chave do produto". O Windows será reativado de forma automática via internet após concluir tudo.

3. Escolha da Versão e Termos

Selecione a versão correta do sistema (ex: Windows 10 Pro / Windows 11 Pro) de acordo com a licença que você possui, aceite os Termos de Licença e avance.

4. O Ponto Crucial: Tipo de Instalação

Sempre clique na segunda opção: "Personalizada: Instalar apenas o Windows (avançado)". É ela que permite limpar o disco antigo e fazer a formatação real do zero.

5. Formatando e Particionando as Unidades

Nesta tela você verá os discos rígidos do PC subdivididos em partições. Para uma instalação limpa completa:

  1. Clique em cada partição do seu Disco principal e clique em Excluir, até restar apenas o "Espaço Não Alocado na Unidade 0".
  2. Com esse espaço limpo selecionado, clique em Novo e depois em Aplicar (o Windows criará partições de sistema automaticamente).
  3. Selecione a partição principal do tipo "Principal" e clique em Avançar para disparar a cópia de arquivos.

O computador irá copiar os dados e reiniciar sozinho algumas vezes. Importante: Quando ele reiniciar a primeira vez, remova o pendrive da porta USB para o computador não entrar no instalador de novo!

Capítulo 5

Configuração Final e Pós-Instalação

O sistema vai ligar na tela de assistente de voz da Cortana ou configurações iniciais do usuário.

Passos Finais do Sistema:

  • Confirme sua região e layout de teclado.
  • Conecte na sua rede Wi-Fi ou cabo de rede.
  • Insira sua conta Microsoft (Outlook/Hotmail) ou crie um usuário local offline.
  • Configure uma senha ou PIN de segurança para login.

Dica de Ouro pós-formatação:

Vá em Configurações > Atualização e Segurança > Windows Update e clique em "Verificar se há atualizações". Deixe o sistema rodar para instalar todos os drivers de vídeo, áudio e rede automaticamente.

Conclusão

Parabéns! Seu computador foi formatado e o Windows reinstalado com sucesso do zero absoluta. Agora você tem uma máquina limpa, mais rápida, livre de vírus e pronta para uso profissional ou pessoal!
